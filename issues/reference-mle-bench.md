## Reference

- Paper: https://arxiv.org/abs/2410.07095
- GitHub: https://github.com/openai/mle-bench

## What it is

MLE-bench evaluates agents on machine-learning engineering tasks, using competition-style artifacts and grading.

## Why it matters for Agent Olympics

ML engineering is a useful advanced event category, but it needs local grading and artifact discipline.

## Design takeaways

- Complex tasks need local evaluation scripts.
- Artifacts and reproducibility matter.
- Cost and compute limits should be explicit.

## Possible follow-up

Defer ML engineering events until core Ops, Code, Wiki, Safety, and Coordination events are stable.
