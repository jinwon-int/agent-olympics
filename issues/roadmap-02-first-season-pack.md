## Purpose

Build the first Agent Olympics 2026 task pack.

## Proposed events

- `ops-001`: Telegram final reply does not appear.
- `ops-002`: Decide whether clean reinstall is needed.
- `code-001`: TypeScript regression fix with targeted tests.
- `knowledge-001`: Convert an incident transcript into a wiki-ready closeout.
- `coord-001`: Merge contradictory node reports into a commander report.

## Acceptance criteria

- Each task has a valid Task Envelope.
- Each task has hidden judge notes or an answer key.
- Each task lists allowed and forbidden actions.
- Each task can be attempted by at least OpenClaw, Hermes, and a CLI agent.
- Each task includes clear required outputs.
